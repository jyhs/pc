
# About

此项目是 vue + element-ui 构建的后台管理系统。

## 技术栈

vue2 + vuex + vue-router + webpack + ES6/7 + less + element-ui


## 项目运行


```
https://github.com/fdjiangwu/Coral-Manage.git 

cd Coral-Manage  

npm install

npm run dev (访问线上后台系统)

访问: http://localhost:8002

```
